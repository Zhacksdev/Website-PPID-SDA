# Redesign Website PPID Sidoarjo

Proyek ini adalah konsep desain ulang (*redesign*) untuk website **PPID (Pejabat Pengelola Informasi dan Dokumentasi) Kabupaten Sidoarjo**.

PPID Sidoarjo merupakan portal utama pelayanan informasi publik yang berfungsi untuk memudahkan masyarakat dalam mengakses dokumen dan informasi secara transparan. Proyek ini bertujuan memberikan tampilan yang lebih segar, modern, dan interaktif melalui pendekatan **Responsive Website Animation**.
